# 🌱 IoT-Based Smart Irrigation & Greenhouse Monitoring System

An IoT-enabled smart irrigation and greenhouse monitoring system developed using **ESP32**, designed to automate irrigation, monitor environmental conditions, and improve agricultural efficiency through real-time cloud monitoring with **Blynk IoT**.

---

## 📌 Overview

This project integrates multiple environmental sensors with an ESP32 microcontroller to continuously monitor greenhouse conditions and automatically control irrigation, ventilation, and safety mechanisms.

The system provides real-time monitoring through the Blynk IoT platform while reducing water consumption and minimizing manual intervention.

---

## ✨ Features

- 🌡️ Real-time temperature and humidity monitoring
- 🌱 Automatic irrigation based on soil moisture level
- 🔥 Fire detection with buzzer alarm
- 💡 Ambient light monitoring
- 🌧️ Rain detection
- 🚰 Water tank level monitoring using ultrasonic sensor
- 🪟 Automatic greenhouse ventilation using servo motor
- ☁️ Remote monitoring via Blynk IoT Dashboard
- 📟 Live sensor values through Serial Monitor

---

## 🛠 Hardware Components

| Component | Purpose |
|-----------|----------|
| ESP32 Dev Board | Main Controller |
| DHT11 | Temperature & Humidity |
| Soil Moisture Sensor | Irrigation Control |
| Flame Sensor | Fire Detection |
| Water Sensor | Rain Detection |
| LDR | Light Measurement |
| Ultrasonic Sensor HC-SR04 | Water Tank Level |
| Servo Motor SG90 | Window Automation |
| Relay Module | Pump Switching |
| Water Pump | Irrigation |
| Buzzer | Alarm |
| LCD I2C | Local Display |

## ⚙️ System Workflow

1. Read sensor data.
2. Analyze environmental conditions.
3. Control irrigation pump automatically.
4. Control greenhouse ventilation.
5. Detect fire and trigger alarm.
6. Upload sensor readings to Blynk Cloud.
7. Display live values on Serial Monitor.
